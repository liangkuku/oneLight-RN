# 开始

> **Note**: 确保安装 node，推荐统一版本 V18.15.0

## Step 1: 安装依赖

安装依赖库

```bash
# 安装第三方库
yarn

# 首次启动项目需执行
cd ios

# 安装捆绑器bundler
bundle install

# 安装 CocoaPods 管理的 iOS 依赖项
bundle exec pod install
```

## Step 2: 启动项目

在安卓和 iOS 模拟器或真机上启动项目:

### For Android

```bash
# 启动安卓
yarn android
```

### For iOS

```bash
# 启动iOS
yarn ios
```
